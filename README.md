# Creati (creati-java-system)

크리에이터/도전자들을 위한 성장 로그 시스템  
실패 기록 → 원인 분석 → 회고 → 재도전 액션플랜까지 연결하는 팀 프로젝트

## Repository Structure
- /docs : 기획/설계/DB/화면설계 문서
- /src  : Java 소스(이클립스 프로젝트 포함 가능)
- /assets : 이미지/로고 등 리소스

## Docs
- docs/requirements.md : 기능명세
- docs/db-schema.md : DB 설계
- docs/ui-wireframe.md : 화면설계

## Branch Strategy
- main: 안정 버전
- feat/*: 기능 개발 브랜치
- docs/*: 문서 작업 브랜치
